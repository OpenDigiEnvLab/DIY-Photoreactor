# DIY-Photoreactor
# This repository provides supplementary information for the project: 
# Developing an innovative, affordable integrated photoreactor system for water treatment applications"
# Authors: Devi Sarai Orozco, César Costa-Vera, Jose Luis Gonzales, Ricardo Araguillin López, Elad Levintal and Thomas Cadenbach. 

# The repository includes: 
# - Guide for 3D printing with parts dimensions and main parameters.
# - Photoreactor case and holders 3D printing files (STL.)
# - Step by step self-assembly guides for the photoreactor system modules:
#   > Main Control Module
#   > Light Module
#   > Temperature and pH
#   > Cooling and Sitrring
#   > Automatic sampling and dosing
# - Arduino code for the automatic sampling/dosing module. 
# - Pictures and videos of the assembled system for visual reference
